# MISSÃO INTERGALÁCTICA: O TESTE DA HUMANIDADE 

## 🎮 Sobre o Projeto
Game intergalático da etzinha
MISSÃO INTERGALÁCTICA é um quiz interativo desenvolvido com HTML, CSS e JavaScript sobre Arquitetura Computacional.

No jogo, Esther foi sequestrada pela comandante alienígena ZyraX e precisa provar que os humanos dominam a tecnologia para salvar a Terra da invasão alienígena.

Cada resposta correta protege a humanidade.  
Cada erro aproxima o planeta da destruição.

---

## 🧠 Conteúdos do Quiz

- CPU
- ULA
- Registradores
- Memória RAM e ROM
- EPROM
- Flash
- DMA
- Barramentos
- Processadores Intel

---

## ⚙️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎮 Funcionalidades

- Sistema de perguntas e respostas
- Contador de pontos
- Sistema de fases
- Mudança de cenários conforme a fase
- Status da Terra baseado nos erros
- Final de vitória ou derrota

---

## 🌍 Status da Terra

| Erros | Status |
|--------|---------|
| 0 | SEGURA |
| 1-2 | ALERTA |
| 3+ | PERIGO |
| 7+ | INVASÃO |

---

## 🚀 Como Jogar

1. Clique em **INICIAR AVENTURA**
2. Responda as perguntas
3. Tente salvar a Terra acertando o máximo possível
